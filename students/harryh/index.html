Design Brief

1. Game Design Document

Fortress Evolution is a futuristic tower defence game where players defend a core against escalating waves using strategic towers, research upgrades, and map control.

2. Code Structure

The game is organised into distinct systems for towers, enemies, projectiles, waves, progression, and UI so new content can be added cleanly.

3. Gameplay Systems

Players place towers on a grid, earn coins from kills, unlock upgrades, and survive wave-based invasions with boss encounters every 10 waves.

4. Balancing Plan

Early waves teach the basics, mid-game mixes control and damage towers, and late-game requires support, economy, and specialised counters to stay competitive.
